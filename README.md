# Sales & Revenue Analysis Dashboard

## Overview

This project analyzes sales and revenue data using Microsoft Excel and presents the results through an interactive dashboard.

## Features

* Total Revenue KPI
* Total Quantity Sold KPI
* Total Orders KPI
* Product Revenue Analysis
* Sales Trend Analysis
* Interactive Filters

## Tools Used

* Microsoft Excel
* Excel Formulas
* Charts and Graphs
* Data Filters
